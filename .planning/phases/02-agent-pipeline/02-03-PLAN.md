2. Add the following functions to `agents/news_agent.py`:

   **FinBERT loader (module level, loaded once at import):**
   ```python
   from transformers import pipeline as hf_pipeline
   _finbert = None  # lazy load to not block import during testing

   def get_finbert():
       global _finbert
       if _finbert is None:
           _finbert = hf_pipeline(
               "text-classification",
               model="ProsusAI/finbert",
               return_all_scores=True,
           )
       return _finbert
   ```
   Use lazy loading so tests can import the module without loading the 500MB model. The first real call loads it (3-5s, acceptable).

   **`async def fetch_finnhub_headlines(tickers: list[str], api_key: str) -> list[dict]`**
   - Use httpx.AsyncClient to GET `https://finnhub.io/api/v1/news?category=general&token={api_key}` (general market news)
   - Also fetch per-ticker news: GET `https://finnhub.io/api/v1/company-news?symbol={ticker}&from={7_days_ago}&to={today}&token={api_key}` for each equity ticker
   - Combine and deduplicate by headline text
   - Each headline dict: `{"headline": str, "source": str, "related": str, "datetime": int, "url": str}`
   - Rate limit: sequential per-ticker fetches with no delay needed (60 req/min, ~10 tickers = 11 requests)
   - Timeout: 10 seconds per request

   **`def filter_headlines_for_tickers(headlines: list[dict], tickers: list[str]) -> list[dict]`**
   - Keep headline if: ticker symbol appears in headline["headline"] text (case-insensitive) OR ticker appears in headline.get("related", "")
   - Tag each kept headline with matching ticker(s) in a "matched_tickers" field
   - Cap: max 5 headlines per ticker
   - Return filtered list

   **`def score_sentiment(text: str) -> float`**
   - Call get_finbert()(text[:512])[0]
   - Extract scores dict: {r["label"]: r["score"] for r in results}
   - Return: scores.get("positive", 0.0) - scores.get("negative", 0.0)
   - Range: [-1.0, 1.0]

   **`def score_headlines(headlines: list[dict]) -> list[dict]`**
   - For each headline, add "sentiment" key with score_sentiment(headline["headline"])
   - Discard headlines where abs(sentiment) < 0.1 (near-neutral, uninteresting)
   - Return scored list

   **`def aggregate_sentiment_by_ticker(scored_headlines: list[dict]) -> dict[str, float]`**
   - Group scored headlines by matched_tickers
   - For each ticker, compute mean sentiment of its headlines
   - Return {ticker: mean_sentiment}

   **`def compute_overall_sentiment(aggregate: dict[str, float]) -> float`**
   - Return mean of all ticker sentiment scores

3. Update `handle_fetch_news` handler's else branch:
