       Tests:

       - `test_sse_connects`: Make a GET request to `/events` with `stream=True`. Assert response status is 200. Assert `content-type` header starts with `text/event-stream`. Close the stream.

       - `test_pna_header`: Make an OPTIONS request to `/events` with headers `Origin: http://localhost:5173`, `Access-Control-Request-Method: GET`, `Access-Control-Request-Private-Network: true`. Assert response contains header `access-control-allow-private-network` with value `true`.

       - `test_trigger_returns_200`: Make a POST request to `/trigger`. Assert status 200. Assert response JSON equals `{"status": "triggered"}`.

       - `test_event_delivered`: Set `ev._fastapi_loop = asyncio.get_running_loop()` and `ev._event_queue = event_queue`. Call `push_event("test", "thought", {"text": "hello"})`. Then `event = await asyncio.wait_for(event_queue.get(), timeout=2.0)`. Assert `event["agent_id"] == "test"`, `event["type"] == "thought"`, `event["text"] == "hello"`.

       NOTE: For SSE streaming tests, use `httpx.AsyncClient(transport=ASGITransport(app=app))`. The SSE endpoint is infinite-streaming, so for `test_sse_connects` use `async with client.stream("GET", "/events") as response:` and check status/headers without trying to consume the full body.

       - `test_agent_addresses_stable_across_imports`: Import all 5 agents, record addresses. Import them again (they're singletons). Assert addresses match. This confirms deterministic seeds produce stable addresses.

       - `test_mock_mode_default_true`: Assert `os.getenv("MOCK_DATA", "true").lower() == "true"` (default when env var not set is true).
