2. Add computation functions to `agents/portfolio_agent.py` (or a separate `agents/services/portfolio.py` if you prefer separation -- at Claude's discretion):

   **`compute_sector_allocation(portfolio: list[dict]) -> dict[str, float]`**
   - Group by "sector" key, sum weights per sector
   - Crypto holdings (type=="crypto") go under sector "Crypto"
   - Return dict like {"Technology": 0.49, "Healthcare": 0.14, ...}

   **`compute_herfindahl(weights: list[float]) -> float`**
   - Formula: sum of squared weights. `float(np.sum(np.square(weights)))`
   - Input: list of weight fractions summing to 1.0

   **`compute_portfolio_beta(equity_tickers: list[str], weights: list[float], lookback_days: int = 365) -> float`**
   - Use yfinance to download historical close prices for equity_tickers + ["SPY"]
   - Compute daily returns via pct_change().dropna()
   - Weighted portfolio returns = (returns[equity_tickers] * weights).sum(axis=1)
   - Beta = cov(port_returns, spy_returns)[0,1] / cov[1,1]
   - Uses numpy `np.cov()`

   **`compute_correlation_matrix(equity_tickers: list[str], lookback_days: int = 90) -> dict[str, dict[str, float]]`**
   - Use yfinance to download historical close prices
   - Compute daily returns, then `returns.corr()`
   - Return nested dict: {ticker: {other_ticker: correlation_coefficient}}
   - Round to 4 decimal places

   **`compute_top_holdings(portfolio: list[dict], n: int = 5) -> list[dict]`**
   - Sort portfolio by weight descending, take top n
   - Return list of dicts with ticker, weight, sector keys
